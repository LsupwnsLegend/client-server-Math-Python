Математический задачник.

Для того чтобы развернуть данный проект, необходимо знать свой IP-адрес или IP-адрес компьютера где будет находится сервер.

Для того чтобы узнать IP-адрес необходимо в командной строке ввести команду ipconfig, среди сетей найти либо адаптер Ethernet, либо адаптер Wi-Fi. Далее найти строку "IPv4-адрес" и скопировать этот адрес.
После того как вы получили адрес:
	1. Открыть папку "server";
	2. Открыть файл "ip.txt";
	3. Заменить IP в файле на IP, который вы получили ранее.
	4. Открыть папку "client";
	5. Открыть файл "ip.txt";
	6. Заменить IP в файле на IP, который вы получили ранее.
Важно чтобы в файле ip.txt не было больше символов, кроме как самого IP-адреса

После того как вы заменили IP-адреса, нужно сначала запустить файл server.exe, если все пройдет хорошо, вы увидите надпись:	
	Server IP: ﻿"ВАШ IP-АДРЕС"
	Server Port: 7000

Если все удалось, можно запустить файл client.exe, как только клиент установит соединение с сервером запустится само приложение. В консоли сервера появится надпись о подключении.
